# Lyfter

Lyfter is a full-stack application with a React frontend and an Express backend.

## Folder Structure

```
lyfter/
├── client/               # React frontend
│   ├── public/
│   └── src/
│       ├── components/
│       ├── pages/
│       ├── assets/
│       └── App.js
├── server/               # Express backend
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── utils/
│   └── server.js
├── .env                  # Environment variables
├── .gitignore
├── README.md
└── package.json
```
